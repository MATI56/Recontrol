## About

Experimental game focused on translating psychological tension into gameplay systems.  
Objects behave unpredictably, creating unstable environments that challenge player control.

## Features

- Physics-driven interactions  
- Emergent gameplay  
- Minimalist design  
- System-based tension  

## Tech

- Engine: Unity
- Language: C#
- Tools: Dotween
